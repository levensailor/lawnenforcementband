# Contributing to Lawn Enforcement

Thank you for your interest in contributing to the Lawn Enforcement website.

## How to Contribute

1. **Fork the repository** and clone it locally.

2. **Create a new feature branch** from `main`:

   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make your changes** and commit them with a clear, descriptive message.

4. **Push your branch** to your fork:

   ```bash
   git push origin feature/your-feature-name
   ```

5. **Open a Pull Request** against the `main` branch. In the PR description, explain:
   - What the change does
   - Why this change should be added to the repository
   - Any testing you performed

## Guidelines

- Keep the retro Windows 95 / 1990s aesthetic consistent
- Test locally with `python main.py` before submitting
- Do not commit `.env` files or any credentials
- Update `CHANGELOG.md` for any feature additions
